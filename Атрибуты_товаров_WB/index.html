Атрибуты товаров WB

На WB ограниченный список атрибутов товаров, которые можно указать для товара. Атрибуты определяются относительно предметов (товарных категории 2-го уровня), к которым относятся товары, поэтому для работы с атрибутами категории товаров уже должны быть загружены из WB. 

  • Доступные атрибуты
  • Загрузка справочников атрибутов WB
    • Справочник ТН ВЭД кодов

Доступные атрибуты​

Просмотреть перечень атрибутов, которые могут быть указаны для товара можно на вкладке Характеристики предмета формы **Рабочий стол - WB - Справочники **(рис. 1).

В начале работы вкладка будет пустая. 

Получить характеристики предметов - по кнопке выполняется запрос и импортируются все атрибуты, которые могут использоваться для всех предметов. 

В верхней части вкладки в таблице Атрибут приведен перечень всех атрибутов в колонке Наименование атрибута и их Коды.  Для каждого предмета выделена колонка, в которой флажками отмечены те атрибуты, которые можно задать для товаров, относящихся к этому предмету.  

В колонке Тип атрибута указано, является значение атрибута строковым, числовым или заполняется из предустановленного справочника:

  • Строковый атрибут -  значение атрибута заполняется произвольным значением, состоящим из символов (в основном слов, допускается использование цифр)
  • Числовой атрибут -  значение атрибута заполняется произвольным числом
  • Тип атрибута соответствует названию атрибута - например, для атрибута Цвет указан тип атрибута Цвет. В таком случае значение атрибутов заполняется из справочников WB. К таким атрибутам относятся:
    • Цвет
    • Пол
    • Страна производства
    • Сезоны
    • Ставки НДС
    • *ТН ВЭД коды *

Удалить из ERP - выделенный атрибут будет удален из системы, но не с портала WB, при повторной загрузке будет снова импортирован.

В нижней части вкладки в таблице Настройки перечислены все предметы, и для тех, для которых выделенный вверху атрибут является обязательным, установлен флаг в столбце Обязательный. Эти данные также импортируются из WB и не могут быть изменены.

Рис. 1 Перечень атрибутов товаров

Загрузка справочников атрибутов WB​

Значения атрибутов Цвет, Пол, Страны производства, Сезоны, Ставки НДС и ТН ВЭД коды заполняются значениями из справочников WB. Справочники загружаются на соответствующих одноименных вкладках формы **Рабочий стол - WB - Справочники **(рис. 2).

Получить список ... - кнопка для загрузки справочника.

Удалить из ERP -  удаляет выделенное значение атрибута из системы lsFusion ERP. При повторном запросе такие удаленные значения атрибутов будут снова импортированы.


Рис. 2 Загрузка предустановленных справочников атрибутов

Справочник ТН ВЭД кодов​

Для каждого предмета (категории) на WB определен перечень кодов ТН ВЭД, к которым могут относится товары. Эти коды загружаются на вкладке  **ТН ВЭД коды **формы **Рабочий стол - WB - Справочники. **По кодам ТН ВЭД также определяется, подлежит ли товар маркировке. Для таких кодов установлен флажок в колонке КИЗ. Если товар относится к коду, для которого установлен флажок КИЗ, то то по заказам с такими товарами нужно отправлять их коды маркировки.

По выделенному курсором коду ТН ВЭД в нижней части отображается предмет, к которому он относится.
