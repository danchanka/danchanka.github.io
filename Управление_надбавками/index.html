Управление надбавками

Для управления надбавками нам необходимо создать специальный вид цены.

Для этого заходим на форму Цены\Справочники\Виды цен

Нажимаем кнопку "Добавить расчетную цену"

Заполняем атрибуты и настройки Расчетного вида цены:

Слева в дереве классификатора можно установить основную надбавку для всех групп товаров или для каждой в частности:

Справа в общем списке товаров, можно установить надбавку для каждой отдельной позиции:

Надбавка % (перегруженная) обычно совпадает с установленным процентом надбавки и используется чаще всего для ограничений надбавок по каким либо условиям.

Нажимаем "Сохранить" или "ОК" для сохранения нашего нового вида цен в системе.
