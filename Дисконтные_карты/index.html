Дисконтные карты: Обзор

В программе поддерживается учет дисконтных карт. 

При создании в системе дисконтной карты она сразу активна, но чтобы по ней применялась скидка, необходимо настроить акцию, в которой данная дисконтная карта будет участвовать. 

Тип дисконтной карты

Дисконтные карты могут быть классифицированы по типам. На форме Розничная торговля - Типы дисконтных карт можно создать необходимое количество типов карт (рис. 1). Для типа дисконтной карты указывают(рис. 2):

  • Наименование - удобное название
  • Код - код типа карты

Рис. 2 Настройка типа дисконтной карты

Дисконтные карты​

Все дисконтные карты в системе отображаются на форме Розничная торговля - Дисконтные карты (рис. 3). В верхней части формы отображается список дисконтных карт, а в нижней - по выделенной дисконтной  карте можно просмотреть чеки и товары чека, которые были приобретены с предъявлением карты. 

Рис. 3 Дисконтные карты

Добавить/Редактировать/Удалить - кнопки для соответствующих действий с дисконтной картой. 

Замена карты - позволяет заменить клиенту существующую дисконтную карту на другую.  При замене карты данные старой карты переносятся в новую (данные владельца, накопленные суммы и бонусы). Новая карта должна быть уже заведена в системе. 

  1. Нажать Замена карты
  2. Ввести/отсканировать код старой карты
  3. Проверить переносимые данные
  4. Ввести отсканировать штрихкод новой карты
  5. Сделать старую карту неактивной (см. описание настроек дисконтной карты)  

Рис. 4 Замена дисконтной карты

Сгенерировать дисконтные карты - позволяет создать в системе несколько новых дисконтных карт.  При нажатии на кнопку откроется окно, где:

Наименование (Нумератор) - нумератор (должен быть настроен), который определяет правила формирования номера дисконтных карт

Количество дисконтных карт - количество карт, которые будут сгенерированы.

Наименование ( Тип дисконтных карт) - программа сгенерирует дисконтные карты указанного типа. 

Процент скидки - процент скидки, который будет указан в дисконтной карте (информационное поле, не влияет на процент применяемой скидки)  

Рис. 5 Создание нескольких дисконтных карт одновременно

Настройка дисконтной карты​

 На форме дисконтной карты настраиваются и отображаются следующие параметры.

Код - код дисконтной карты

Наименование - произвольное наименование дисконтной карты

Начальная сумма - начальная сумма накоплений по карте, если указать сумму, она будет добавлена к общей сумме накоплений по карте. Сумма накоплений равна сумме стоимости покупок в применением карты. Сумма накоплений отображается по каждой карте на форме Розничная торговля - Дисконтные карты (рис. 3)

Процент скидки - текстовое поле, не влияющее на применение скидки по карте. Скидка применяется в соответствии с условиями акций, в которых участвует карта

Тип карты - тип к которому относится дисконтная карта. 

Неактивный - если отметить признак, карта будет неактивна, система не будет распознавать ее для применения скидок.

Дата выдачи/окончания действия - период действия дисконтной карты. Дата выдачи обязательно должна быть указана, по умолчанию дата выдачи заполняется датой создания карты. 

Полное имя/Имя/Фамилия/Телефон/E-mail/Адрес/День рождения - данные держателя карточки 

Сумма переводов - если карточка выдана взамен старой(ых), то здесь будет указана сумма переведенных на карту накоплений.

Номер дисконтной карты для перевода - если эта карта была заменена на  другую, здесь будет указан номер карты, на которую заменили текущую. 

Сумма перевода - если эта карта была заменена на  другую, здесь будет указана сумма накоплений, переведенная на новую карту.

Вид цен для дисконтных карт - для дисконтной карты можно указывать отдельный вид цены, которая будет использована на кассе вместо обычного вида цены для розничных продаж, указанного для кассы. 

Бонусы - в этой секции доступна информация по начисленным, использованным и заблокированным бонусам. 

Вкладка Акция -  здесь отображаются все акции, которые настроены с использованием дисконтной карты. Те акции, в которых участвует текущая дисконтная карта отмечены в колонке В акции 

Вкладка Начисление бонусов (рис.7)  -  На данной вкладке отображаются бонусы, начисление которых производилось вручную пользователем, либо если эта карта была заменена на  другую, здесь будет указаны бонусы, которые были переведены с другой карты. 
Чтобы начислить бонусы необходимо нажать кнопку Добавить и в появившейся строке указать количество бонусов, затем сохранить изменения.
