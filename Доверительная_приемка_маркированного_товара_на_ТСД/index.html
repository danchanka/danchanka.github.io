Доверительная приемка маркированного товара на ТСД

Доверительная приемка означает, что на приходе вы сканируете не все марки поступившего товара, а только определенный процент. Предполагая, что если отсканированная часть марок валидные, то и остальные тоже.  

Настройка​

Чтобы использовать доверительную приемку, нужно установить для товарной группы или отдельного товара процент доверительной приемки. Если для группы и товара из этой группы установлено разное значение, то приоритет будет иметь значение, установленное на уровне товара. 

  • на уровне товарной группы на форме Справочники - Товар - Товарные группы откройте карточку товарной группы и на вкладке Параметры в поле Процент доверительной приемки установите значение в процентах, которое будет действовать на все товары в этой группе;


    *Рис. 1 Процент доверительной приемки на уровне группы *

  • на уровне товара на форме Справочники - Товар - Товары открыть карточку товара и на вкладке Основные в поле Процент доверительной приемки установите значение в процентах.

      
    Рис. 2 Процент доверительной приемки на уровне товара

Для типа документа ТСД, по которому будет проводиться доверительная приемка, на форме Оборудование - Терминалы сбора данных - Группы типов документов нужно выделить документ, вызвать контекстное меню , нажать команду Показать расшифровку и отметить флаги: Маркировка, Разрешить ввод новой марки,  Разрешить доверительную приемку. 


Рис. 3 Настройка флагов типа документа ТСД

Приемка​

На приемке в документ ТСД :

  1. Отсканируйте штрихкод товара.
  2. Введите его количество.
  3. Откроется окно сканирования марок.
  4. Сканируйте марки на товарах до тех пор, пока количество марок не появится кнопка Проверено, это значит, что число отсканированных марок достигло установленного процента доверительной приемки.
  5. Нажмите кнопку Проверено, все указанное в п.п. 2 количество будет добавлено в документ.


Рис. 4 Добавление в документ ТСД товара по доверительной приемке

Необходимое для сканирования количество марок рассчитывается следующим образом: для указанного по товару количества система высчитывает установленный процент доверительной приемки и округляет вверх до целого числа. В примере на рис. 4 установлен процент доверительной приемки 20%, количество по строке товара - 12шт.  Количество марок, которые нужно отсканировать получается 3 (12*20%=2,4 +округление вверх = 3).

При формировании приходной накладной в нее импортируются и соответственно станут на приход только марки, отсканированные в документ ТСД.
