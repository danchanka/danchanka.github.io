Сборка и разборка товаров

Для создания простых комплектов из товаров, например, подарочных наборов или обратного их разукомплектования, используют документы сборки и разборки. 

1. Товар.​

В справочнике нужно завести карточку товара, который получается в результате сборки или разборки. Такой товар ничем не отличается от других товаров и имеет тип SKU товар.

2. Сборка ​

Чтобы составить комплект из нескольких товаров, нужно на форме Склад - Сборки товаров создать новый документ кнопкой Добавить.

На форме нового документа нужно:

  1. Заполнить шапку:
    указать Операцию, 

    Настройка операции Сборка

    Основные параметры операции определяются на вкладке Ограничения.

    Должно быть заполнено количество  - если отмечено, количество в документе должно быть больше 0, иначе будет сообщение о запрете :"Для текущей операции обязательно заполнение поля "Количество" положительным значением"
    Должна быть заполнена цена -  если отмечено, цена должна быть больше ,0 иначе будет сообщение о запрете :"Для текущей операции обязательно заполнение поля "Цена""
    Должны быть строки у документа - если отмечено, система не позволит провести Сборку, если в ней не будет строк, будет сообщение : "Для текущей операции обязательно наличие строк спецификации"
    Запретить внесение более 1 позиции для сборки - если отмечено, система не позволит провести документ Сборки, если в нем будет более одной строки, будет сообщение "Для текущей операции запрещено внесение более 1 позиции для сборки"
    Запретить внесение товара не из своей подгруппы товара - если отмечено, программа не позволит провести документ если 4 уровень товарной группы товара не соответствует 4 уровню товарной группы товара в который указан в сборке в части Товар, т.е. составные товары входящие в сбор и товар сборный должны быть одной товарной группы при этой галочке, иначе будет сообщение: "Для текущей операции запрещено внесение внесение товара не из своей подгруппы товара"
    Единицы измерения должны совпадать - если отмечено, программа не позволит провести документ если единица измерения товара в строке не соответствует единице измерения товара в который указан в сборке в части Товар, будет сообщение: "Для текущей операции единицы измерения должны совпадать"
    Запретить расхождение по кол-ву - если отмечено, программа не позволит провести документ если во всех строках сборки, у товаров единица измерения такая же как у товара указанного в части Товар, сравнивается количество этого товара и суммированное количество товаров по всем строкам, если оно отличается, то будет сообщение :"Для текущей операции запрещено расхождение по кол-ву в документе и в строках"
    Запретить одинаковые товары - при этой галочке не позволит провести Сборку если в строках сборки и в части Товар будет указан один и тот же товар, будет сообщение: "Для текущей операции запрещены одинаковые товары в документе и в строках"

    Склад, на котором находятся комплектующие и происходит сборка. Собранные комплекты будут оприходованы на этот же склад.
    Штрихкод/Код/Наименование - выбрать из справочника товар, который получится в результате сборки.
    Кол-во - указать какое количество товара будет собрано. 

  2. Заполнить спецификацию, т.е. добавить в спецификацию товары, из которых состоит собираемый комплект, и указать количество, которое необходимо для сборки указанного количества комплектов. 

  3. Проверить цены. Для каждого товара в спецификации программа подтянет учетную цену и рассчитает сумму исходя из количества. Простым суммированием учетных цен компонентов будет получена учетная цена комплекта.

  4. Провести документ. В результате с остатков по складу будут списаны компоненты и поставлены на учет собранные комплекты.    

3. Разборка​

Разборка - процесс обратный сборке, когда один товар разбирают на составляющие. Разборка применяется, например, для фасовки товара в более мелкую тару, или разукомплектации наборов и т.д. Для создания разборки на форме Склад - Разборки товаров нужно создать документ по кнопке Добавить. 

На форме нового документа нужно:

  1. Заполнить шапку:
    указать Операцию,
    Склад, на котором находится товар, который будут разукомплектовывать. Полученные после разборки товары будут оприходованы на этот же склад.
    Штрихкод/Код/Наименование - выбрать из справочника товар, который будет разобран.
    Кол-во - указать какое количество товара будет разобрано. 
  2. Заполнить спецификацию, т.е. добавить в спецификацию товары, которые получатся в результате разборки, и указать количество, которое получится в результате разборки указанного количества товара в шапке. 
  3. Проверить цены. В спецификации для каждого товара, который будет получен в результате разборки, можно указать, какой Процент цены от  исходного товара на него относится, на основании чего программа рассчитает  учетную цену товара. 
  4. Провести документ. В результате с остатков по складу будут списаны первоначальные товары, а товары, полученные в результате разборки, — поставлены на учет.
