Создание заказа на производство изделия

В новом производственном заказе  необходимо заполнить данные. 

 Операция - выберите подходящую операцию, в зависимости от предстоящего процесса производства. 

Склад компонент - склад, с которого будут расходоваться материалы для производства. 

Склад изделий - склад, на которые будут приходоваться готовые изделия. 

Создать документ производства - если свойство включено, при проведении документа будет создан документ Производство и готовая продукция поступит на указанный склад изделий в количестве, указанном в столбце Кол-во (произведено) на вкладке Изделия.

Создать документ списания сырья - если свойство включено, при проведении документа будет создан документ списание на количество сырья, фактически  использованного для производства количества изделий, указанных в  столбце Кол-во (произведено) на вкладке Изделия.

Дата действия с/ по - срок действия производственного заказа.

На вкладке Изделия необходимо создать список изделий, которые будут произведены по данному заказу. Для этого можно использовать кнопку Добавить и вводить изделия по одному или вкладку **Подбор. **Необходимо указать количество изделий, планируемое  производству.

На вкладке Компоненты необходимо создать список используемого для производства сырья. 

Заполнить компоненты из спецификации - будет создан список компонентов в соответствии со спецификациями, указанными на вкладке Изделия.  При этом, если в спецификации в качестве компонента используется изделие, в созданном списке будет указано данное изделие, а не его компоненты.

Заполнить из спецификации (рекурсивно по компонентам) -  будет создан список компонентов в соответствии со спецификациями, указанными на вкладке Изделия.  При этом, если в спецификации в качестве компонента используется изделие, в созданном списке будут указаны его компоненты.

На вкладке Замена отображаются остатки компонентов с учетом настроенных замен. При необходимости использовать замену,  галочкой надо указать, какой именно  товар замены использовать, этот товар будет в дальнейшем использован для калькуляции стоимости. 

На вкладке Цены калькулируется стоимость производства изделий.  В блоке SKU перечислены те компоненты, с учетом замен, которые будут использованы в производстве. В колонке Кол-во отображена потребность в сырье, Текущий остаток показывает наличие сырья и есть ли необходимость пополнения склада. Цену производимых изделий можно формировать используя кнопки: 

  • Исп. max цену - будут использованы цены партии с максимальной ценой.
  • Исп. min цену- будут использованы цены партии с минимальной ценой
  • Исп. цену большего остатка - будут использованы цены партии с большим остатком 

На закладке Партия можно выбрать для каждого компонента конкретную партию  и нажать Использовать партию, или нажать кнопку Партия с мах остатком (для всех) , для всех компонентов будут проставлены такие партии и использованы цены их этих партий. 

Также на вкладке Компоненты можно использовать кнопку Заполнить партии (ФИФО) и будут применены цены из проставленных партий. 

После того, как определены партии и цены для расчетов на вкладке Изделие необходимо нажать Заполнить цены для изделий и программа заполнит данные по себестоимости и розничной цене готовых изделий в заказе. 

Если вы используете заказ в том числе как документ производства (свойство Создать документ производства , то проверьте значение количество произведено. По умолчанию оно будет равным плану производства, но вы можете изменить количество. Если вы ориентируетесь на объем продаж, то можно использовать кнопку Заполнить из остатков и программа заполнит количество произведено исходя из дневных продаж по складу, указанному как склад изделий.
