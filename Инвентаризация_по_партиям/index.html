Инвентаризация по партиям

Любая  инвентаризация может проводиться по партиям. Для этого в операции инвентаризации (или в настройках документа) должен быть отмечен признак По партиям. 

Не проверять наличие партий для строк описи - нужно отметить в том случае, если опись товара создается без указания партий, и партии расписываются автоматически при заполнении сличительной ведомости. Признак становится доступным (только в настройках операции), если отмечено По партиям.

 
Рис. 1 Настройки операции инвентаризации

При проведении инвентаризации по партиям опись можно создавать несколькими способами:

  • Работа с описями на ТСД с указанием партий
  • Опись на ТСД без партий
  • Опись по списку SKU
  • Опись по остаткам

Работа с описями на ТСД с указанием партий​

Чтобы создавать опись на ТСД с указанием партий нужно:

  1. Должны быть установлены настройки для использования партий на ТСД

    Чтобы выгружать партии товаров в ТСД. Для этого  на форме Администрирование - Настройки на вкладке Оборудование в поле Количество дней для выгрузки партий установить период в ТСД, за который в ТСД будут выгружаться партии товаров. 


    *Рис. 1а Настройка выгрузки партий *  

    Для документа ТСД, который формирует опись, отметить флажок **Включить партионный учет **(Оборудование - Терминалы сбора данных - Группы типов документов команда Показать расшифровку в контекстном меню поля Флаг типа документа) 


    Рис. 2а настройка отображения партий в документах ТСД

  2. При загрузке данных в ТСД включить тумблер **Загружать партии товаров **


    *Рис. 2 Загрузка партий на ТСД *

  3.  При сканировании в документ товара на ТСД нужно:

    1. указать количество,
    2. нажать кнопку Расписать по партиям,
    3. на ТСД отобразятся все загруженные партии товара, для каждой нужно установить количество остатка,
    4. нажать ОК.

      Рис. 3 Создание описи с партиями на ТСД
      Изменять расписанное по партиям количество на остатке можно подняв на редактирование позицию документа на ТСД.

  4. На вкладке Описи инвентаризации, нажмите кнопку Заполнить из документов ТСД, затем выберите из списка нужный документ и Сохраните опись (рис. 4). В опись будет добавлена страница с товарами с указанными партиями. (рис. 5)

    Рис. 4 Добавление документа ТСД в опись


    Рис. 5 Готовая страница описи с партиями

  5. Сделайте сличительную ведомость.

Опись на ТСД без партий​

  1. В настройках операции должен быть отмечен признак Не проверять наличие партий для строк описи(отображается если в настройках операции стоит признак по Партиям).

  2. На ТСД создается документ со списком товаров и их фактического количества и импортируется в инвентаризацию.

    *Рис. 6 Импорт описи ТСД *

  3. Сформируется страница описи без указания партий товаров. Если есть необходимость, то партии можно проставить вручную: кликнуть в поле Партия и выбрать подходящую из открывшегося списка. 


    Рис. 7 Страница описи без указания партий товаров

  4. На вкладке Сличительные ведомости нужно нажать Заполнить остатки и система автоматически распишет остатки по партиям по LIFO. Результат расписывания остатков по партиям можно посмотреть на вкладке Партия в колонке Кол-во по описям. Если результат нужно откорректировать, то по тем позициям, где партию надо изменить, нужно вручную указать партию в описи (см. п.п. 3)


    Рис. 8 Результат распределения остатков по партиям

Опись по списку SKU ​

При создании описи по списку SKU  партии товаров можно заполнить по тому же алгоритму, как при работе с описями по документам ТСД без партий: при заполнении сличительной ведомости система автоматически распишет партии по методу LIFO (рис. 8) (результат отобразится на вкладке Партии сличительной ведомости),  а при необходимости изменить результат автоматического распределения остатков по партиям, нужно внести партии вручную в описи. 

Опись по остаткам​

Если создавать страницу описи по кнопке Создать страницу описи с остатками, то система автоматически добавит в опись все товары в наличии по выбранным группам с указанием партий. Если нужно внести изменения, партию можно изменить вручную.


Рис. 9 Опись по остатком с заполненными партиями
