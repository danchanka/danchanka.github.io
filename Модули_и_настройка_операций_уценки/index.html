Модули и настройка операций уценки

Модули​

Markdown, Set10Markdown

Настройки операции уценка​

Ограничение по уценке товарных групп​

Можно отметить группы, на товары из которых не будет действовать текущая операция. При сохранении документа уценки будет отображаться предупреждение и документ не будет сохранен.

Ограничения​

Запрет создания документов без указания номинала стикера и количества - такой документ система не даст сохранить.

Можно отметить номиналы, которые будет запрещено использовать в операции.

Параметры​

Печатать QR-коды без просмотра - отправлять сразу на принтер, без открытия печатной формы.
