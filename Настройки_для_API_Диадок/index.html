Настройки для API Диадок

Для настройки корректной работы электронного документооборота необходимо настроить API интеграцию с контуром Диадок и внести в систему данные УКЭП , ввести данные доступа к своей учетной записи в Диадок и настроить в своей учетной записи (ящике) Диадок свою компанию и организации контрагентов. 

Настройка интеграции

В Администрирование – Интеграция вкладка Диадок API необходимо указать

Url - адрес доступа к контуру Диадок

Ключ разработчика - ключ API доступа, который получают в процессе регистрации

Логин пользователя Диадока/ID Ящик Диадок - заполняются для возможности подписывать документы пользователями системы без личного ящика в Диадок.  С заполненными настройками пользователи с доступом к формам и командам подписи документов Диадок могут их подписывать. 

Рис. 1 Настройки API доступа​

Электронная подпись

Подписание документов электронной подписью гарантирует их подлинность и юридическую силу. Подпись документов происходит через CryptoPro.

В Администрирование – Интеграция вкладка УКЭП необходимо ввести Имя и Пароль ключа усиленной квалифицированной электронной подписи (УКЭП) для каждой компании, которая будет участвовать в ЭДО. Вверху формы можно указать данные, которые будут использоваться по умолчанию.

Рис. 2 Настройка УКЭП​

Данные доступа к ящику Диадок

На форме Документы - Пользователи Диадока с помощью кнопки Добавить необходимо создать пользователя Диадока и указать Логин и Пароль доступа в ящик Диадок.    

Рис. 3 Список пользователей Диадока​

Рис. 4 Данные доступа пользователя в ящик Диадок​

Далее карточку сотрудника, который будет осуществлять действия по ЭДО необходимо связать с пользователем Диадока.  Карточку сотрудника открываем кнопкой Редактировать на форме Справочники - Сотрудники. 

В поле Пользователь Диадока из выпадающего списка надо выбрать соответствующий ящик и  Сохранить изменения в карточке. 

Рекомендуется заполнить .ФИО и должность сотрудника. Эти данные будут автоматически заполнятся при подписи и отправке документов.

Рис. 5 Привязка пользователя Диадока к сотруднику​

В системе можно настроить неограниченное количество учетных записей Диадока, однако к пользователю можно привязать только один ящик, с которым он будет работать.

Организации

Собственная компания и контрагенты, участвующие в ЭДО отображаются на форме Документы - Организации Диадока.

Чтобы загрузить данные о собственной организации необходимо нажать кнопку Получить мои организации. 

Чтобы загрузить данные контрагентов, надо нажать Получить данные контрагентов организации. 

Чтобы увидеть подробные данные об организациях надо выделить организацию курсором и нажать Просмотреть.

Рис. 6 Список организаций ​

Рис. 7 Карточка организации Диадока​

Данные об организациях, которые участвуют в ЭДО, включая данные собственной компании, можно настроить только в Диадоке.  

Если информация об организации изменилась, ее необходимо изменить в Диадоке, а затем снова использовать кнопки **Получить (...) **для обновления данных об организациях.

Единственное поле, которое можно редактировать из программы - Тип организации. По умолчанию в Диадоке все организации являются "Юридическим лицом, состоящим на учете в налоговых органах". Поле Тип организации можно изменить или в карточке организации, или в списке организаций.

Рис. 8 Изменение типа организации​

В карточке организации (компании) нужно указать организацию Оператор ЭДО и **ИД участника **
ЭДО - идентификационный номер организации-участника ЭДО, который присваивает оператор ЭДО. 

Рис. 9 Данные по ЭДО организации​

Для организаций Диадока необходимо указать соответствующие организации справочника lsFusion. Только для организаций с установленными соответствиями автоматически формируются документы УПД/УКД.

Рис. 10 Заполнение соответствия организаций Диадок и lsFusion.​

Организация - оператор ЭДО

Организация, которая является оператором ЭДО, должна быть заведена в справочнике организаций. Для нее обязательно должен быть указан признак Явл. оператором ЭДО и указан ИД оператора ЭДО - идентификационный номер организации оператора ЭДО. Список доверенных операторов юридически значимого электронного документооборота можно найти на сайте ФНС России.
