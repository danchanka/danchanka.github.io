Печать акционных ценников

Виды акционных ценников, как правило, разрабатываются для торговых объектов индивидуально, и обычно на акционном ценнике указана предыдущая и действующая, сниженная, цена. 

Образец:


Рис. 1 Акционный ценник 

Базовый механизм печати акционных ценников предполагает, что если у товара есть акционная цена, то при печати ценников отдельным файлом будут сформированы  акционные ценники (отдельный файл на каждый тип ценника). Для корректной работы данного механизма необходимо выполнить ряд настроек. 

1. Для регулярных и акционных цен должны быть заведены разные виды цены. Товары расцениваются операцией расценки.

2. На форме Розничная торговля - Ценники - Типы ценников необходимо создать акционный ценник и для него указать:

  • в поле Вид цен -  вид системной цены, обычно "Розничная(расценка)" 
  • в поле Управленческий вид цен - акционный вид цены
  • на вкладке Склады отметить отделы магазинов, для которых должен быть доступен ценник

    Рис. 2 Форма типа ценника
  • на вкладке Товары не отмечать товары в поле Вкл. (если отметить, то акционный ценник будет ценником по умолчанию и система будет формировать и отправлять на печать его для всех товаров, независимо от наличия акционной цены)
     
    Рис. 3 Вкладка Товары

3. В акционном прайсе должен быть указан настроенный тип акционного ценника в поле Тип ценника. Это можно сделать непосредственно в прайсе. Или в настроенной для акционного прайса операции на вкладке Параметры. Цена до, указанная в прайсе, выводится на акционный ценник перечеркнутой. 


Рис. 4 Акционный прайс


Рис.5 Настройки операции прайса

4. Чтобы при отправке на печать ценников для акционного товара не печатались стандартные ценники, нужно в настройках типа стандартного ценника указать в поле Управленческий вид цен  акционный вид цены, а в операции прайсов для регулярной цены указать тип стандартного ценника.

Рис. 6 Настройки типа ценника

Рис. 6 Настройки операции прайса регулярной розничной цены

Печать акционных ценников производится согласно общим правилам печати ценников.По кнопке Распечатать ценники формируются отдельные файлы:

1). с ценниками, установленными для товара по умолчанию;

2). с настроенными в системе акционными ценниками. 

По кнопке перепечатать ценники можно выбрать любой из настроенных в системе типов ценников и распечатать.

Формы, из которых можно распечатать акционные ценники:

  • Рабочий стол - Расценка (РФ)/Переоценка (РБ)

  • Розничная торговля - Акты расценки (РФ) / Акты переоценки (РБ)
