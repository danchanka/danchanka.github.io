Категории товаров

На WB предусмотрен двухуровневый справочник категорий товаров. Список категорий как 1-го, родительского, так и 2-го уровня предопределён и не может быть дополнен собственными категориями продавца. Первый уровень товарных категорий называется Родительские категории товаров, категории второго уровня называются Предметами. Приступая к работе с WB категории товаров нужно загрузить в первую очередь. 

Родительские категории товаров​

Родительские категории товаров  отображаются на одноименной вкладке формы **Рабочий стол - WB - Справочники (рис.1). **В начале работы форма будет пустая.

Получить родительские категории товаров - загружает родительские категории из WB. На форме отобразятся Наименования и Коды категорий, а в столбце Активный отмечены действующие категории.  

Для каждой загруженной категории доступна кнопка Получить список предметов родительской категории, по которой из WB можно загрузить категории второго уровня, т.е. предметы выделенной родительской категории. Предметы также можно загрузить все сразу на вкладке Список предметов, которая описана ниже. После загрузки предметов по каждой родительской категории их количество отобразится в колонке Кол-во предметов, а предметы выделенной курсором категории можно будет просмотреть в нижнем блоке Список предметов. 

Удалить из ERP - удаление выделенной родительской категории из системы. Категория не будет удалена на WB, поэтому при повторной загрузке удаленные категории снова будут импортированы. 


Рис. 1 Вкладка Родительские категории товаров

Предметы​

Предметами называются дочерние (2-го уровня) категории родительских категорий товаров. Они отображаются на вкладке Список предметов формы **Рабочий стол - WB - Справочники **(рис. 2). В начале работы форма будет пустая.

Получить список предметов - по кнопке загружается список предметов (дочерних категорий) всех загруженных родительских категорий. По тем родительским категориям, которые не загружены или удалены из системы lsFusion ERP, предметы не загрузятся.

По каждому загруженному предмету на вкладке отображается его Код, Наименование, Наименование и Код родительской категории товаров. 

Для каждого предмета на WB определён ряд атрибутов, которые задаются либо из справочников WB, либо произвольно продавцом. Перечень и значения таких атрибутов также загружаются из WB. В справочнике предметов можно для выделенного предмета по кнопкам Получить ТН ВЭД коды и Получить характеристики предмета загрузить соответствующие данные. Справочники с такими данными по всем предметам можно загрузить на соответствующих вкладках, подробно атрибуты предметов описаны в статье Атрибуты товаров WB. Перечень загруженных ТН ВЭД кодов и список атрибутов по выделенному курсором предмету можно просмотреть в нижнем блоке на вкладках **ТН ВЭД код ** и Атрибут.
