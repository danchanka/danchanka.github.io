Перемещение товаров между ячейками

Для перемещения товаров между ячейками предусмотрена возможность планирования перемещений с помощью Заказа на перемещение по ячейкам и непосредственно документ Перемещение по ячейкам. 

Заказ на перемещение по ячейкам​

Заказ на перемещение представляет собой план перемещения, на основе которого далее формируется документ перемещения. 

Заказы отображаются на форме WMS - Заказы на перемещение по ячейкам. Для создания нового используется кнопка Добавить. 

 

На форме заказа необходимо заполнить данные:

Операция - выбрать операцию перемещения

Склад - выбрать склад/отдел магазина на котором будут перемещаться товары

Ячейка (из)/(в) - указать откуда и куда перемещается товар

Спецификация - заполнить спецификацию товарами, которые нужно переместить.  указать их количество. Ячейка(из)/(в) заполнится автоматически в соответствии с шапкой заказа, однако эти данные можно изменить по необходимости. 

На вкладке Дополнительные параметры можно установить время автоматического закрытия заказа

На вкладке Исполнение можно фиксировать исполнителя заказа, время начала и окончания выполнения. 

Заполненный заказ нужно провести. Он создаст резерв перемещения

Перемещение по ячейкам​

Перемещение по ячейкам - это документ, который уменьшает остаток товара в ячейке, откуда перемещается товар (Ячейка (из), и ставит это количество на остаток в ячейку, куда перемещается товар (Ячейка(в). 

Перемещение по ячейкам можно создать на основе заказа на форме  WMS - **Заказы на перемещение по ячейкам **на вкладке Действия кнопкой Перемещение. Все необходимые данные в документе будут заполнены в соответствии с заказом. Необходимо только проверить их и провести документ. Перемещение будет выполнено.

Все перемещения по ячейкам отображаются на форме WMS - Перемещения по ячейкам. Для создания нового перемещения используют кнопку Добавить. 

Для перемещения необходимо заполнить данные:

Операция - выбрать операцию перемещения

Склад - выбрать склад/отдел магазина на котором будут перемещаться товары

Ячейка (из)/(в) - указать откуда и куда перемещается товар

Спецификация - заполнить спецификацию товарами, которые нужно переместить.  указать их количество. Ячейка(из)/(в) заполнится автоматически в соответствии с шапкой документа, однако эти данные можно изменить по необходимости.

На вкладке Основание отображаются заказы, на основании которых сформировано перемещение. также заказы указаны для каждой строки перемещения. 

Если перемещение создано не по заказу, строки в спецификацию можно добавить связав перемещение с заказом или с несколькими:

  1. нажать Включить заказы в перемещение
  2. отметить заказы
  3. согласиться создать строки по выбранным  заказам 

Созданный документ нужно Провести для соответствующего изменения остатков по ячейкам.
