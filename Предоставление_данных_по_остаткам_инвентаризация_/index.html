Предоставление данных по остаткам (инвентаризация)

Предоставление сведений об остатках на 01.12.2024​

Сведения об остатках прослеживаемого товара необходимо предоставить в  МНС по состоянию на 01.12.2024. Вносить корректировки в сведения по остаткам допускается до 01.02.2025.

Чтобы предоставить в налоговую сведения об остатках прослеживаемого товара, необходимо создать документ инвентаризации и отправить его в МНС. 

Инвентаризация должна содержать только товары, подлежащие прослеживаемости, и быть обязательно по партиям и с указанием марок, если товар подлежит маркировке. 

В операции инвентаризации обязательно должны быть установлены параметры:

  • По партиям
  • Марки
  • Импорт из ТСД в 1 опись

Остальные параметры инвентаризации выбираете те, которые подходят для вас, Тип инвентаризации выбираете любой, признак Частичная тоже по желанию, если хотите использовать. Напомним, что признак Частичная доступен только для типа инвентаризации "Товары по остаткам". См. Инвентаризации и Инвентаризация по партии

 

Создайте инвентаризацию только по товарам, которые подлежат прослеживаемости.  После проведения документа инвентаризации на форме Склад - Инвентаризации появится кнопка Отправить в МНС.

1. Нажмите кнопку Отправить в МНС для отправки данных. В инвентаризации должны быть только прослеживаемые товары. Если инвентаризация содержит не прослеживаемые товары, то отображаются предупреждение и инвентаризация не будет отправлена

2. Приложение NTClientSoftware предложит выбрать сертификат (ключ ЭЦП).3. Приложение видит все сертификаты на Вашем ПК и предложит один из них. Если это не сертификат Вашей ЭЦП, нажмите  Больше вариантов
4. Из списка сертификатов выберите свой сертификат ЭЦП и нажмите ОК. Далее введите пароль и документ отправится.

После выгрузки инвентаризации, на вкладке Отправка в МНС будут отражены данные по выгруженному документу, строкам документа и маркам документа.

Корректировка сведений об остатках​

После того, как сведения об остатках отправлены в МНС, их можно корректировать. 

Необходимо внести изменения в отправленной инвентаризации или в соответствующих справочниках.

Например, если необходимо изменить количество товара, нужно изменить данные в описи, заново заполнить сличительную ведомость, сохранить и провести инвентаризацию.  А если необходимо изменить, например, страну происхождения товара или его наименование, то это нужно сделать в карточке товара и сохранить изменения. 

После внесения изменений необходимо заново отправить инвентаризацию в СПТ кнопкой Отправить корректировку в МНС.  Каждый корректирующий документ будет отображаться на вкладке Отправка в МНС в блоке Документы выгрузки в МНС, а данные по строкам и маркам с изменениями отображаются в блоке Строка документа/марки документа выгрузки в МНС по выделенному документу выгрузки.
