Карточка товара

Чтобы открыть карточку товара, необходимо в Справочники - Товары дважды кликнуть на нужный товар, или выделить его и нажать кнопку Редактировать. Если нажать кнопку Добавить, вы создадите новый товар (откроется пустая карточка товара для заполнения).  Кнопка Копировать создает копию выбранного товара, это позволяет быстро создать новый товар со схожими характеристиками, вам остается лишь поменять некоторые данные и сохранить карточку.

В карточке товаров хранятся все значимые  характеристики товаров, которые распределены по соответствующим вкладкам. Ряд вкладок и характеристик доступен только при подключении определенного функционала. Например, если интеграция с  ФГИС Меркурий подключена, то в карточке товара в основных данных отображаются поля Код номенклатуры и Номенклатура Ветис, а если подключена интеграция с ЕГАИС, то есть и вкладка ЕГАИС (рис. 2).


Рис.1. Справочник товаров.

Вкладка Основные данные​

На вкладке  Основные данные указаны базовые свойства, такие как Название, Бренд, Производитель, Единица измерения, Поставщик и т.д. Какие из доступных в программе атрибутов товара будут отображаться и настраиваться в карточке товара определяют настройки на уровне товарной группы, а также подключенные модули.   

Автоматическое формирование наименования товара​

Для структурирования имен товаров (ввод наименования товара по определенным правилам) предусмотрен механизм автоматического формирования наименования товара исходя из заданных атрибутов товарной группы. Данный механизм настраивается на форме Справочники - Товар - Настройка атрибутов и подробно описан в статье Атрибуты. Суть механизма в том, что для необходимых атрибутов задается порядок их отображения наименовании товара. 

 
Рис. 2. Формирование наименования товара.

Вкладка Штрихкоды​

Для товара поддерживаются множественные штрихкоды, которые  можно просмотреть на вкладке Штрихкоды, добавить новые или деактивировать неактуальные. 

Чтобы создать новый штрихкод можно нажать кнопку Добавить и ввести штрихкод в появившуюся новую строку, либо нажать кнопку Сгенерировать штрихкод, и появится новая строка, в которой уже будет внесен созданный программой штрихкод.  Каждый штрихкод может использоваться для различных видов упаковки товара, данные вносятся в соответствующие колонки. Для каждого штрихкода могут быть отмечены следующие признаки:

Активный - штрихкод используется в системе.

Основной - штрихкод по умолчанию используется в документах при выборе товара.

Множитель -  используется для перевода базовой единицы в количество в упаковке, если поставка товара осуществляется не в базовых единицах. 

Не загружать в оборудование - если отмечен признак, штрихкод не будет загружаться в кассы, весы, и др. оборудование

Штрихкод для закупки - используется в документах закупки.

Штрихкод для продажи - используется в документах продажи.

Чтобы удалить штрихкод необходимо нажать кнопку Удалить в строке штрихкода.

Вкладка Классификаторы​

На вкладке Классификаторы отображаются группы, к которым принадлежит товар. Группа основного классификатора указана в поле  Полное имя группы. Чтобы включить товар в какую-либо группу дополнительного классификатора, необходимо выделить классификатор в списке, кликнуть в поле Каноническое имя, выбрать группу, в которую включить товар и нажать ОК. 


Рис. 4. Привязка товара к дополнительному классификатору.

Вкладка Региональные параметры​

На вкладке Региональные параметры собраны атрибуты, зависящие от места торговли товаром - Страны :

  • Код ТН ВЭД;
  • ставка НДС - ставка НДС, которая применяется к товару. Ставка выбирается из справочника налогов и отображается с учетом выбранной Даты. При указании НДС нужно руководствоваться исключительно товаром, а не тем, работает поставщик этого товара с НДС или без. 
  • принадлежность к группе естественной убыли;
  • нормы отходов на колбасные и мясные изделия.

Здесь же указывается принадлежность товара к статистическим классификаторам, путем выбора узла дерева в колонке Каноническое имя.


Рис. 5 Вкладка Региональные параметры

Вкладка Цены​

На вкладке Цены перечислены все цены расчетного и пользовательского вида. Если для текущего товара были созданы прайсы (вкладка Прайс), то в блоке Пользовательский вид цены, в колонке Текущая цена будут значения из прайсов. Если для товара существует Базовая цена, то отобразится Расчетная цена, как результат умножения Базовой цены на % надбавки. Признак Цена с НДС указывает, что в Расчетную цену также добавлен НДС. Цены отображаются относительно выбранного склада/магазина.


Рис. 6 Цены товара

Вкладка Изображение​

На вкладке Изображение можно загрузить изображение товара, которое будет видно в справочнике при выборе товара, может быть передано на ТСД, прайс-чекер и в другое оборудование, выгружено на сайт и т.д. Для загрузки используют команду Загрузить, для удаления загруженного изображения - команду Сбросить. 


Рис. 7 Вкладка Изображение

Вкладка Прайс​

Здесь отображаются прайсы, в которых есть данный товар. Подробнее о прайсах.


Рис. 8 Вкладка Прайс

Вкладка Ценники​

Здесь отображаются все ценники, которые настроены в системе. В колонке Вкл. отмечают ценники, которые используются по умолчанию для товара. отсюда можно сформировать ценники для печати по отдельности нажав на кнопку Сформировать ценник или все сразу по кнопке Сформировать ценники. Будут сформированы ценники с указанием цен по выбранному складу/магазину. Подробнее о ценниках и этикетках.
