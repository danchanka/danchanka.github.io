Интернет-заказ

Для обработки интернет-заказа его необходимо открыть для редактирования (кнопка Редактировать).
Загруженный из CMS сайта интернет-заказ уже заполнен информацией о покупателе, товаре, данных для доставки, форме оплаты и т.д.


Рис.  1 Интернет-заказ

Поля шапки интернет-заказа​



Название

Назначение области

1

Статус

Статус интернет-заказа определяется в большинстве случаев автоматически на основании изменения статусов строк заказа. Первоначально заказ имеет статус «Новый»

2

Склад поставщика

Склад, с которого будет производиться доставка интернет-заказа. Может быть один склад, с которого доставляются все интернет-заказы, либо склад может определяться CMS сайта и передаваться в программу.

3

Номер дисконтной карты

Номер дисконтной карты, если она использовалась покупателем при оформлении заказа.

4

Оформить карту

При отметке данного признака для покупателя будет заведена дисконтная карта в программе и доставлена вместе с заказом.

Тип оплаты

В данном поле указывается тип оплаты заказа покупателем, возможные варианты:

Наличными

ЕРИП

Оплата наличными или картой по умолчанию предполагает оплату при получении товара покупателем. Оплата ЕРИП является предоплатой

16

Сумма оплаты (нал)/(безнал)/(ЕРИП)

Указывается сумма произведенной оплаты в соответствующем поле. Заполняется после получения информации о произведенной доставке (кроме ЕРИП)

17

Перевозчик

Указывается перевозчик, который доставляет заказ и принимает оплату

18

Стоимость доставки

Указывается стоимость услуг по доставке данного заказа.

Заказанные товары отображаются в спецификации интернет-заказа с некоторыми особенностями.


Рис.  2 Спецификация интернет-заказа

  • Для каждой единицы товара формируется отдельная строка, т.е. если заказано несколько единиц одного и того же товара, в спецификации будет несколько строк.
  • Цена и Цена со скидкой импортируются из CMS сайта интернет-магазина, а Процент скидки программа высчитывает на основании импортированных данных.
  • По умолчанию программа назначает товары (партии товаров) в заказе со склада поставщика, указанного в шапке заказа. Для информации в блоке Склады отображаются данные о наличии данного товара на других складах, и при необходимости можно вручную выбрать другой склад для назначения товара. Кроме того, может быть выполнена индивидуальная доработка механизма назначения товаров по нескольким складам.
  • Для каждой строки заказа в результате первоначального назначения товаров присваивается статус Назначено, либо Нет в наличии. Далее по мере работы с заказом статус каждой строки изменяется и влияет на статус всего заказа.

После успешной обработки интернет-заказа оператор переводит заказ в статус Согласован и проводит его.
