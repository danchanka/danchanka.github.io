Проверка статуса марки на ТСД

При работе с ТСД реализована возможность проверки статуса марок. Проверять марки можно через ГИС "Электронный знак" или через сервис «Цифровой помощник кассира».

Через ГИС "Электронный знак" можно получить информацию по маркам СИ и УКЗ. Информация о статусе зарубежных кодов маркировки (например, марки ГИС МТ "Честный знак")не предоставляется.

Через сервис «Цифровой помощник кассира» можно получить информацию как по СИ и УКЗ, так и по КИЗ (зарубежным маркам)

Для проверки статуса марки на главном экране приложения lsFusion BCS нужно отсканировать марку. Результат отобразится на экране:

Рис. 2 Статус марки УКЗ

Проверка через ГИС "Электронный знак"​

Для того, чтобы воспользоваться такой возможностью нужно иметь регистрацию в ГИС "Электронный знак" и настроить подключение для ТСД в системе:

  1. На форме Оборудование - Конфигурация ТСД отметить признак Вкл. для Параметра Опции приложения Группы Основные настройки
  2. Дважды кликнуть в поле Значение, чтобы открылось окно для ввода значений параметров
  3. Внести следующие данные
    • GIS_enable=true
    • GIS_server=https://api.datamark.by - путь api подключения к серверу ГИС ЭЗ
      GIS_user= user@mail.by - логин входа в кабинет пользователя ГИС ЭЗ
      GIS_pass=password - пароль для входа в кабинет пользователя ГИС ЭЗ
      GIS_timeout=10000 - таймаут соединения

Рис. 3 Настройка подключения к ГИС "Электронный знак" для проверки марок

На терминале эти настройки находятся в меню Настройки - Основные настройки - Опии приложения. Примите во внимания, что если настроить опцию на устройстве, то при обновлении базы на ТСД настройки перезапишутся теми, которые установлены в системе.


Рис. 4 Настройки подключения на терминале

Проверка через сервис «Цифровой помощник кассира»​

Для проверки статуса через "Цифровой помощник кассира" нужно зарегистрироваться в сервисе и настроить подключение в конфигурации ТСД.

Обратите внимание!​

Если в Конфигурация ТСД настроено подключение к ГИС "Электронный знак", то для использования сервиса "Цифровой помощник кассира" нужно для параметра GIS_enable установить значение true.

  1. На форме Оборудование - Терминалы сбора данных - Конфигурация ТСД отметить признак Вкл. для Группы параметров Маркировка товаров
  2. поле Значение заполнить значения для следующих Параметров:
  • A1. URL - путь подключения к сервису ( для продуктива)
  • А1. Имя пользователя - логин подключения к сервису
  • А1. Онлайн проверка марок - установить значение Yes
  • А1. Пароль - пароль подключения к сервису


Рис. 5 Настройка подключения к сервису "Цифровой помощник кассира" для проверки марок

На терминале эти настройки находятся в меню Настройки - Маркировка товара. Примите во внимания, что если настроить опцию на устройстве, то при обновлении базы на ТСД настройки перезапишутся теми, которые установлены в системе.

Рис. 6 Настройки подключения на терминале

Обращаем Ваше внимание, что ГИС "Электронный знак " планирует вывод из эксплуатации и прекращение поддержки API-методов POST /labels/ и POST /v3/labels, использующихся в настоящее время для запроса сведений о маркированном товаре.
