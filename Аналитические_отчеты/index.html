Аналитические отчеты

На форме BI – Аналитические отчеты (mobile) представлены отчеты по розничным продажам в разрезе отделов магазинов.

На вкладке Продажи представлена информация по сумме выручки и скидки по магазинам. 

На вкладке Покупатели представлена информация по количеству чеков продажи и возврата по магазинам. 

На вкладке Средний чек представлена информация по сумме среднего чека по магазинам. 

На вкладке Таблица  представлена сводная информация по отчетам в форме таблицы с отображением итоговых суммарных значений по всем магазинам. 

В шапке формы выводятся итоговые значения по отображаемым данным.

Внизу формы отображаются фильтры значений. При выборе фильтра он подсвечивается тем же цветом, которым отображаются значения в отчете.


Рис. 1 Форма Аналитические отчеты

Аналитические отчеты параметризуются по периоду отбора данных и по магазинам.

Чтобы установить период, за который необходимо составить отчет, необходимо кликнуть в поле Период и выбрать один из предопределенных периодов, или установить даты начала и окончания периода для отчета. 

Для выбора магазинов, по которым необходимо получить данные, надо нажать кнопку  и отметить требуемые магазины. Отмеченные записи подсвечиваются фиолетовым.

Для удобства отображения магазинам можно назначить краткие наименования на форме отдела магазина (Розничная торговля - Справочники - Магазины) в поле **Наименование mobile **(рис. 3).
