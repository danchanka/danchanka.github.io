Товарные группы: Обзор

В программе предусмотрен один основной справочник товарных групп, а также может быть создано несколько дополнительных справочников товарных групп.

Основные группы товаров​

На форме Справочники - Товар - Товарные группы (рис.1) формируется структура основного товарного классификатора. Группы могут иметь неограниченное количество уровней подгрупп (вложенность). Для создания подгруппы, необходимо установить курсор на группу, которая будет являться родительской для новой подгруппы и нажать  кнопку Добавить.

**Перенести отмеченные группы в текущую - используется для изменения иерархии товарных групп. В колонке Отм. отмечаются группы (подгруппы), которые нужно переместить, после чего курсор устанавливается на группу, которая станет родительской для отмеченных групп. По нажатии кнопки Перенести отмеченные группы в текущую изменяется положение отмеченных групп(ы) в иерархии, группы перемещаются вместе с товарами. Изменения вступают в силу после нажатия кнопки Сохранить.

Удалить - используется для удаления группы из иерархии. В колонке Отм*.* отмечаются группы (подгруппы), которые нужно удалить. По кнопке Удалить группы удаляются из дерева категорий, а товары, которые принадлежали удаленным группам, останутся  справочнике Товаров без принадлежности к товарным группам. Их можно будет просмотреть, если использовать фильтр Без групп. Изменения вступают в силу после нажатия кнопки Сохранить.


Рис. 1. Товарные группы.

Каждая группа является определенным сводом правил для входящих в нее товаров. 

На форме товарной группы вводят Наименование группы, Родительский объект, т.е. товарную группу на порядок выше в иерархии, в которую входит данная. Можно задать Код группы и Порядок, в котором будет отображаться данная группа среди групп своего уровня. Также можно сделать группу Неактивной.

Рис. 2 Форма товарной группы

Каждая группа является определенным сводом правил для входящих в нее товаров. На вкладке Параметры можно настроить:

  • Нумератор штрихкодов (для весового товара) *- *шаблон для формирования штрихкодов на товар, продаваемый через весы.

  • Нумератор штрихкодов (для штучного товара) - шаблон для формирования штрихкодов на товар, продаваемый поштучно.

  • Учитывать продажи в наличии товара - расчет среднедневных продаж происходит в дни, когда товар был на остатке.

  • Печатать цену за килограмм(литр) - дополнительная цена на ценнике.

  • Категорийный менеджер- ограничение товаров для сотрудника, при формировании заказов на закупку.

    подробнее

    Сперва указывается на товарную группу менеджер, например Иванов И.И. по группе пиво
    Далее Справочники/Сотрудники/Иванов И.И. вкладка Менеджер - выбрать Иванов И.И.
    Если несколько сотрудников занимаются этой товарной группой, в карточке каждого такого сотрудника во вкладке Менеджер - тоже выбирать Иванов И.И.
    По данному примеру у сотрудника будет  ограничение по товарной группе - доступна группа только пиво
    Где  это ограничение отобразится:
    0) Справочники/Товары

    1. форма 'Ассортиментные матрицы'
    2. форма 'Организация' вкладка признаки разделения
    3. форма 'Заказ (закупка)' - будут ограничены товарные группы

  • Мин. объем/вес для печати цены за кг/л -  если в карточке товара из данной группы будет указан вес/объем ниже указанного здесь, то для этого товара на печать не будет выводится цена за кг/л

  • Тип маркировки - тип кодов маркировки, применяемый к товарам данной группы

  • Обязательное указание марки - если отмечен признак, то без указания марок невозможно будет провести приходные и расходные документы с товаром из этой группы, а также пробить чек  при использовании модуля управления продажами. 

  • Процент доверительной приемки -  параметр для приема маркированного товара в упаковке.

На вкладке **Атрибуты ** определяется список отображаемых атрибутов товаров. Те свойства, которые отмечены галочкой, будут отображаться в карточках товаров данной группы и ее подгрупп.

Рис. 3 Вкладка Атрибуты

На вкладке Значения по умолчанию для некоторых атрибутов товара данной группы можно задать значения, которые будут подставляться в карточку нового товара или при назначении имеющемуся товару данной группы. Здесь же можно указать соответствие текущей группы группам дополнительных классификаторов - на вкладке Дополнительные классификаторы, в поле  Дополнительная группа (по умолчанию).


Рис. 4 Вкладка Значения по умолчанию

По такому же принципу происходит привязка к НДС - вкладка НДС, нормам естественной убыли - вкладка Группа естественной убыли и статистическим группам - вкладка Статистический классификатор. По нажатию кнопки Назначить всем товарам группы, всем товарам будут присвоены дополнительные группы, которые связаны с основным классификатором. При необходимости, пользователь может изменить дополнительные группы в карточке товара.
